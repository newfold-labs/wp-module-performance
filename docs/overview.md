# Overview

## What the module does

**wp-module-performance** provides caching functionality for WordPress sites: page cache and object cache (Redis via credis). It registers with the Newfold Module Loader and is used by brand plugins (e.g. Bluehost). It integrates with wp-module-htaccess and wp-module-installer for cache rules and setup.
